O que é JavaScript?

É uma linguagem de programação, interpretada, para desenvolvimento web que é executada no navegador do cliente em um sandbox, que limita o acesso aos recursos do sistema operacional, fornecendo interfaces predefinidas, a forma de acesso a recursos do sistema se faz por meio de objetos globais disponibilizados ao navegador, como: document, history, local storage e console. Ela foi criada para adicionar dinamismo e interatividade nas páginas.A tipagem é dinâmica e forte, permitindo que variáveis possam ser atribuídas e reatribuídas a qualquer momento, sem a necessidade de declarar seu tipo previamente.Os tipos de dados nativos da linguagem são: Number, String, Boolean, Object, Array e Function.
