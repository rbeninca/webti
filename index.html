A tag para exibir uma imagem é a tag "img" tem atributos como src= que permite definir o caminho para fonte de imagem

Tag de listagem

O HTML possui listas ordenadas e não ordenadas que podem ser definidas com as tags ol e ul respectivamente. A lista odenada permite definir o tipo de sequência por meio do parametro TYPE=[1,A,a,I,i]

Exemplo de lista ordenada

  1. pera
  2. banana
  3. uva
  4. maça
  5. abacate
  6. tangerina

Exemplo lista não ordenada

  • Carne
  • Água Ardente
  • Linguiça
  • Pão de Alho
  • Refrigerante
  • Água com Gás
