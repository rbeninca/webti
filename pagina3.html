O CSS é um arquivo de estilização em cascata criado em 1993, popularizado em 1996, para estilizar e formatar arquivos HTML e CSS através de cores, estilos, fontes, transições, etc...

O CSS evoluiu com o tempo, surgindo as versões 1 2 e 3

  1. CSS 1 1996 - Propriedades básicas de estilização e formatação, controle de posicionamento com display, float e position
  2. CSS 2 1998 - Adição de animação e transição, além de melhoria de position
  3. CSS 3 2008 - Incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex (2008) e grid (2013)

Podemos incluir código CSS de 3 formas distintas

CSS Inline
Quando definimos código CSS de estilização no próprio elemento HTML como:
<span style="color:red">Texto de cor vermelha com CSS inline</span>
CSS Incorporado
Quando definimos código CSS por meio da tag <style>
CSS Externo
Quando definimos código CSS de estilização em outro arquivo: <link rel="stylesheet" href="styles.css">
